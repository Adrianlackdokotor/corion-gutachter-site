# Corion Gutachter Website

## Overview
A German-language website for Corion Gutachter, a vehicle damage assessment and appraisal service. The site includes multiple pages (home, blog, contact, services, forms) with PHP backend for form processing and chat logging.

## Project Architecture
- **Language**: PHP 8.2, HTML, CSS (Tailwind via CDN), JavaScript
- **Server**: PHP built-in development server on port 5000
- **Pages**: index.html, blog.html, contact.html, formular.html, servicii.html
- **Backend**: process-form.php (contact form), save-chat.php (AI chat logs)
- **Uploads**: `uploads/` directory for user file uploads
- **Chat Logs**: `chat_logs/` directory for AI chat conversation logs
